## Installation

```bash
$ npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## CI/CD

```bash
# install packages
$ npm ci

# delete old build
$ npm run prebuild

# build
$ npm run build

# start in production mode
$ npm run start:prod
```

> API URL: http://localhost:57300
